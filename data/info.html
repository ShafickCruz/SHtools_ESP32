Informações Gerais

Projeto SHtools
Este projeto visa fornecer uma interface de controle para dispositivos ESP32, permitindo atualizações OTA, monitoramento serial remoto e execução de comandos remotos.
Funcionalidades
  • Atualização OTA
    • Firmware
    • LittleFS
    • Checksum
    • Rollback
  • Monitoramento Serial remoto
    • Execução de Comandos remotos
